Egy Python 3.11.3-as verzióban írt automatizált program.

Szükséges elemek:
	Python 3.11.3
	Firefox böngésző

Esetleges hibák esetén:

1) Manuális pip installálás:
pip install PyQt5
pip install selenium
pip install pyautogui
pip install opencv-python
pip install beautifulsoup4
pip install numpy
pip install pillow
pip install requests
pip install keyboard
pip install matplotlib

2) Indítás
A Files mappán belül található egy Gui.py fájl, ez a fájl a fő indítóelem.
